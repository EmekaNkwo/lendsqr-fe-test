# SIMPLE ADMIN APPLICATION

## Overview

This is a Dashboard application that displays users and users details. The application is built using Typescript, React, SCSS and Redux and deployed to Heroku

## Logs

- Created atomic reusable components like button, card, form, input, layout, typography, etc
- I passed data using props validation using Typescript interfaces and types for better type checking
- Used SCSS for styling
- I fetched data using React virtualization concept
- Used Redux Toolkit for global state management

## Features

- Added dynamic routing to get all users listing and used useParams to get user details
- Provided a responsive sidebar to provide easy navigation to all pages

## Get Started

- Clone the repository
- Run `npm install` to install dependencies
- Run `npm start` to start the application
- Run `npm run test` to run test on some components
